# Bash Scripting Guide

## Introduction
Bash (Bourne Again SHell) is a command language interpreter widely used in Linux and Unix environments.  
Bash scripts allow automation of tasks, configuration management, and quick prototyping.

---

## Basics

### Running a Script
```bash
#!/bin/bash
echo "Hello, World!"
```

Save as script.sh, then run:
chmod +x script.sh
./script.sh

### How to Use backend.sh
1) clone the repo or copy the file content and make a new fila and save with extension .sh
2) Then Give permission to the file by:
3)   ``` chmod +x backend.sh```
4) Then run the backend.sh file:
        ```./backend.sh```
5) Full backend dependecies will automatically be installed along with folder structure..
